 Statistics-Application

 The Statistics Application can take input from a CSV or XLSX file and can run several statistical computations on the file. 
 
 These statistics are
 
   -> Linear Regression
   
   -> Multiple Regression
   
   -> Studnet T-Tests
   
  -> ARIMA
